# 命名规范

## 1. 通用规则

1. 语义优先，避免无含义缩写。
2. 一个概念只使用一种拼写。
3. 代码标识符统一使用英文。

## 2. 前端

- 组件：`PascalCase`（例如 `WorldCanvas.tsx`）
- Hook：`useXxx`（例如 `useKeyboardShortcuts.ts`）
- 页面：`XxxPage.tsx`
- 样式：`kebab-case` 或模块聚合文件

## 3. 后端

- 路由：`*.routes.ts`
- 控制器：`*.controller.ts`
- 服务：`*.service.ts`
- 中间件：`*.middleware.ts`

## 4. API 与事件

- REST 资源路径使用复数名词：`/api/worlds/:worldId/scenes`
- Socket 事件命名：`domain:entity:action`
- 权限常量集中维护，禁止硬编码散落。

# 07 生活职业与造物

## 职责

生活职业与造物模块承载锻造、工程、炼金、附魔等生活职业，以及魔法物品网格、强化组件、配方和制造任务。

## 入口文件

- `shared/types/life-crafting.ts`
- `docs/世界模板实现蓝图.md`
- 未来后端建议位置：`server/src/services/life-crafting.service.ts`
- 未来前端建议位置：`client/src/world/components/crafting/*`

## 当前约定

- 人物每提升 1 级，可选择 1 级生活职业。
- 生活职业像冒险职业一样有等级、分类、特性和制造范围。
- 魔法物品使用类似背包英雄的空间网格。
- 稀有度越高，可用网格越大。
- 强化组件占用不同形状和空间，安装后提供效果。
- 组件效果最终复用能力系统的效果表达式。

## 待拆分点

- 生活职业定义表。
- 角色生活职业进度表。
- 魔法物品基底和成品表。
- 组件形状校验器。
- 背包式网格编辑器。
- 制造任务和材料消耗规则。

## 验证

- 当前只有共享类型骨架。改共享类型时跑 `npm run build -w server` 和 `npm run build -w client`。

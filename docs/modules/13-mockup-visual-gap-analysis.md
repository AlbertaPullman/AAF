## 2. 左侧列详细差距

### 2.1 命运刻度盘
**Mockup**：
- 132×132px 圆环
- 外圈金色实线边框 + 4px 白色外光晕
- 内圈虚线金色环
- 中心 30px 大数字 `5/8` + 小字章节名
- 顶部 "命 运 刻 度" 间距字
- 底部 5 个圆形按钮（hover 显示）

**当前实现**：
- 大尺寸 SVG 圆环
- 三按钮 + 输入框 + 删除按钮
- 永久可见控制

**重写策略**：完全重写 DOM 和 CSS，参照 mockup HTML L1069-1084

### 2.2 队伍状态面板
**Mockup**：
- `.pc-card` 白→淡蓝渐变卡片
- 左侧 3px 蓝色边框（自己卡用金色）
- 32px 圆形字母头像（渐变背景）
- HP/MP 条带 label + 数值（白色描边）
- `.pc-deploy` 部署生物子卡（虚线边框 + 弯钩连接线）

**当前实现**：
- 简单文字列表
- 无头像
- 无 HP/MP 条
- 无部署生物

**重写策略**：完全重写，参照 mockup HTML L1094-1133

### 2.3 在线玩家面板
**Mockup**：
- 每行：`<b>主/玩/观</b>` 角色徽章 + 玩家名 + 角色名
- 右侧 4 段竖条信号（30/55/80/100% 高度）
- good/mid/bad 颜色区分

**当前实现**：
- 只有玩家名
- 文字延迟显示

**重写策略**：重写 JSX，添加角色徽章和 4 段信号条

## 3. 底部 HUD 详细差距

**Mockup**（最大差距）：
- 80×80px 大圆形头像（左侧独立）
- 主面板左侧凹槽与头像对接（"火炬奖杯"造型）
- SVG 描边只描上/右/下三边，左侧凹槽不描
- 4 排 × 10 列技能格子（第一排带快捷键 1-0）
- 右下角 4 个切换态图标
- 可拖拽浮动定位

**当前实现**：
- 底部固定横条
- 小头像 + HP/MP 条
- 单排 hotbar 按钮

**重写策略**：完全重写，参照 mockup HTML L1247-1311

## 4. 顶部战斗序列条差距

**Mockup**：
- fixed 顶部居中
- 56×80px 立绘卡片
- 当前卡 scale(1.1) + 金色 box-shadow 脉冲
- 左侧金色 round 徽章
- 右侧 "下一回合 ▶" 按钮

**当前实现**：
- 有序列条但样式简单
- 卡片尺寸不同
- 脉冲效果不明显

**重写策略**：调整 CSS，增强脉冲动画

## 5. 右侧系统面板差距

**Mockup**：
- 10 个 1:1 方形 tab（lucide 图标）
- active 金色渐变背景
- badge 红点徽标
- 内容区：res-toolbar + res-tree 多级目录
- 目录树：caret 折叠 + tag-foe/npc/ok 着色

**当前实现**：
- tab 已有图标和 1:1 比例
- 内容区简单列表

**重写策略**：重写内容区，添加工具栏和多级目录树

## 6. 舞台浮动元素差距

**Mockup 有，当前缺失**：
- `.float-win` 能力库窗口
- `.float-win` 能力结算台
- `.float-win` 状态 Tooltip
- `.aoe` AOE 范围圈
- `.dmg-float` 伤害飘字
- `.tool-fab` 左侧工具栏（未在截图中）

**重写策略**：CSS 骨架已就位，需添加 JSX 渲染逻辑

## 7. 实施优先级

按视觉差距大小和用户体验影响排序：

1. **P0 - 底部 HUD**（差距最大，最核心）
2. **P0 - 左侧队伍面板**（差距大，高频查看）
3. **P1 - 左侧命运刻度盘**（差距中等）
4. **P1 - 左侧在线面板**（差距中等）
5. **P2 - 顶部战斗序列条**（已有基础，需优化）
6. **P2 - 右侧系统面板内容区**（结构差距）
7. **P3 - 舞台浮动元素**（功能性，非核心视觉）

## 8. 技术约束

- 保持 React props 接口稳定，不破坏现有调用
- 100% 使用 token 驱动颜色，不写硬编码 hex
- 所有新 CSS 写入 `world-components.css` 或独立模块
- 每完成一个组件立即验证构建

## 9. 预估工作量

- 底部 HUD：2-3 小时（DOM + CSS + SVG 路径）
- 队伍面板：1-2 小时（卡片 + 头像 + 条 + 部署子卡）
- 命运刻度盘：1 小时（圆环 + 按钮）
- 在线面板：30 分钟（徽章 + 信号条）
- 战斗序列条：30 分钟（CSS 调整）
- 系统面板：1-2 小时（工具栏 + 目录树）
- 浮动元素：1-2 小时（JSX 集成）

**总计：7-12 小时**

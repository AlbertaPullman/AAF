# 聊天启动指南

用于在新会话中快速接手 AAF 项目开发。

## 1. 启动前必读

1. `世界需求与填写清单.md`
2. `AI上下文记忆文档.md`
3. `docs/命名规范.md`
4. `docs/interface-registry.json`
5. `docs/接口契约.md`

## 2. 开始开发前输出

1. 当前目标与范围
2. 受影响文件列表
3. 计划执行命令（构建/测试/校验）

## 3. 每次改动后同步

1. 更新 `AI上下文记忆文档.md`
2. 新增接口或事件时更新 `docs/interface-registry.json`
3. 新增命名规则时更新 `docs/命名规范.md`

## 4. 禁止事项

1. 未经评估直接修改核心规则字段。
2. 跳过权限校验直接开放高风险接口。
3. 把跨模块逻辑硬塞进单一文件。

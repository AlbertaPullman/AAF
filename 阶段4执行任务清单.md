# 阶段 4 执行任务清单 - 大厅与世界管理

## 1. 阶段完成状态

**✅ 核心链路已完成** - 2026-03-12

本阶段目标：用户登录后可在大厅完成世界创建、世界浏览、加入世界，并可进入世界页。

## 2. 后端实现完成情况

### 2.1 世界服务 (server/src/services/world.service.ts)

✅ **已完成**
- `createWorld`：
  - 校验世界名
  - 支持 PUBLIC / PASSWORD 创建
  - PASSWORD 模式使用 bcrypt 哈希口令
  - 事务内自动创建 owner 的 WorldMember（GM）
  - 事务内自动创建默认 Scene（sortOrder=0）
- `listPublicWorlds`：返回公开可见世界（PUBLIC / PASSWORD）与 owner、统计信息
- `listMyWorlds`：返回当前用户已加入世界和我的角色
- `joinWorld`：支持公共世界加入、密码世界口令验证、重复加入幂等处理
- `getWorldDetail`：按可见性和成员关系做访问判断，返回 myRole/canJoin

### 2.2 世界控制器 (server/src/controllers/world.controller.ts)

✅ **已完成**
- `createWorld`
- `listWorlds`（支持 `scope=mine`）
- `joinWorld`
- `getWorldDetail`

关键点：
- 统一返回 `success/data/error/requestId` 结构
- 鉴权失败统一 401
- 常见业务错误区分 400/403/404

### 2.3 世界路由 (server/src/routes/world.routes.ts)

✅ **已完成**
- GET `/api/worlds`
- POST `/api/worlds`
- GET `/api/worlds/:worldId`
- POST `/api/worlds/:worldId/join`

所有路由都受 `authMiddleware` 保护。

### 2.4 总路由挂载 (server/src/routes/index.ts)

✅ **已完成**
- 新增 `/worlds` 路由挂载。

## 3. 前端实现完成情况

### 3.1 大厅页面 (client/src/pages/lobby/LobbyPage.tsx)
